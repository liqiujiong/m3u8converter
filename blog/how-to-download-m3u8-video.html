Have you ever found a video on a website but couldn't download it directly? Many sites deliver video using M3U8/HLS, which splits media into many small segments. This guide shows the easiest ways to save and convert those streams.

What Is an M3U8 Video?

M3U8 is a playlist format used by HLS (HTTP Live Streaming). In simple terms:

  • M3U8 file: a text playlist that lists segment URLs
  • TS segments: short video chunks, usually 2-10 seconds each
  • Why platforms use it: adaptive bitrate, better CDN caching, harder direct download

Step 1: Get the M3U8 URL

Before conversion, first extract the M3U8 link from the page.

Method 1: Browser DevTools (Recommended)

  1. Open the webpage with the target video
  2. Press F12 (or right-click and choose Inspect)
  3. Go to the Network tab
  4. Filter requests by m3u8
  5. Play the video and watch the request list
  6. Copy the request URL ending with .m3u8
ğŸ’¡ Tip: If multiple playlists appear, choose one named like index / master or with the larger response size.

Method 2: View Page Source

  1. Open source code with Ctrl+U
  2. Search .m3u8 with Ctrl+F
  3. Copy the full playlist URL

Step 2: Convert to MP4

After copying the M3U8 URL, convert it with our online tool:

  1. Open M3U8 Converter
  2. Wait until the conversion engine is ready
  3. Paste your M3U8 URL
  4. Select output format (MP4 recommended) and quality
  5. Click Start Conversion
  6. Download the converted file when processing is done

FAQ

Q: Why can't some links be downloaded?

Some platforms add encryption or access control. If you see CORS issues, you may need another extraction method or proper request headers.

Q: Why is download speed slow?

Speed mainly depends on the source server and your network conditions. Our tool downloads segments in parallel to improve performance.

Q: Can I choose video quality?

Yes. If the source is a variant playlist, the tool can pick a higher-quality stream, and you can also select output quality settings.
